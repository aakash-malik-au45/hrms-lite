# HRMS Lite – Full Stack Assignment

HRMS Lite is a lightweight Human Resource Management System built as part of a full-stack coding assignment.

The application allows an admin to:

- Manage employees (add, view, delete)
- Track daily attendance
- View attendance history
- See a dashboard summary of today’s workforce status

The focus of the project is clean architecture, RESTful APIs, database persistence, validation, and a usable production-style interface.

## 📂 Features Implemented

### Employee Management
- Add new employee
- Unique employee ID & email validation
- List employees with pagination
- Delete employee

### Attendance Management
- Mark attendance (Present / Absent)
- Update attendance for same employee/day
- View attendance per employee
- Date filtering

### Dashboard
- Total employees
- Present today
- Absent today
- Uses latest attendance per employee for the day
